# Battle Log Analyzer

Учебный проект по книге «Python. К вершинам мастерства».

## Цель проекта

Создать систему для хранения и анализа журнала боевых событий.

Проект позволяет на практике изучить работу с моделями данных,
неизменяемыми объектами, коллекциями, последовательностями и тестированием.

## Что тренируется

- `dataclass`
- неизменяемые объекты с `frozen=True`
- перечисления `Enum`
- `set` и `frozenset`
- аннотации типов
- валидация данных
- переопределение `repr`
- автоматическое сравнение объектов
- тестирование с `pytest`

## Реализовано

- типы боевых событий `EventType`;
- неизменяемая модель `BattleEvent`;
- проверка корректности входных данных;
- защита тегов от изменения;
- тесты модели.
