# Fluent Python Practice

Лусиану Рамальо.

## Задания

### Chapter 1 — Task Board

Собственная коллекция задач с поддержкой:

- индексирования и срезов;
- итерации и обратной итерации;
- фильтрации и сортировки;
- проверки уникальности идентификаторов;
- аннотаций типов;
- тестов pytest.
